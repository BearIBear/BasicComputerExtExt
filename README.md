# BasicComputerExtExt
Вайбкоднутый форк BasicComputerExt с некоторыми исправлениями и улучшениями:
1. Компилятор теперь корректно выдаёт ошибку, если у команды JUMP указана несуществующая метка
2. Подсветка в компиляторе теперь полностью подстроена под команды БЭВМ. До этого она использовала синтаксис процессоров x86, из-за чего не выделяла некоторые команды (JUMP, например)
3. Над регистром CR теперь показывается, какая команда исполняется
4. Сильно улучшена производительность и добавлена возможность менять задержку исполнений команд через F11/F12
5. Различные улучшения для интерфейсов ВУ: теперь они корректно отображаются на macOS, в них был улучшен выбор задержки, а также в имени окна теперь указывается, какие именно регистры ВУ находятся по какому адресу
6. Таблица микрокоманд теперь доступна в графическом интерфейсе

### Загрузка программы
Загрузка программы позволяет быстро загружать ваши программы из .txt файла. Числа, оканчивающиеся на "a", обозначают адрес, с которых начинается ввод данных в ячейки памяти. 
Пример txt файла, который сначала загрузит данные в ячейки с адресами 0x479-0x47E, а потом в 0x713-0x718:
<pre>
<b>479a</b>
0200
EE16
AE12
0C00
EE04
0100

<b>713a</b>
AC01
F309
6E0B
CE01
AE02
EC01
</pre>

### Трассировка
Чтобы выполнить трассировку, укажите адрес начала программы и по желанию задайте задержку, а потом нажмите "Выполнить трассировку". Полученную трассировку можно экпортировать в csv или xlsx файл
![Трассировка](https://github.com/BearIBear/BasicComputerExtExt/blob/master/bcomp-ng-extext-tracing.png)

### Микрокоманды
Таблица интерпретатора БЭВМ встроена в графический интерфейс и позволяет удобно редактировать память микрокоманд, включая их метки. При нажатии на ПКМ можно также откатить до изначальных значений выбранные вами микрокоманды
![Таблица микрокоманд](https://github.com/BearIBear/BasicComputerExtExt/blob/master/bcomp-ng-extext-microcommands.png)
